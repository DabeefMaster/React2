# SPA React Router - CineScope

Aplicación SPA para explorar películas usando React y React Router DOM.

## Rutas

- `/` -> Películas populares
- `/movie/:id` -> Detalle de película
- `/search` -> Búsqueda por título (`q`)

## Configuración

Crea un archivo `.env`:

```bash
VITE_TMDB_API_KEY=tu_api_key_de_tmdb
```

## Ejecutar

```bash
npm install
npm run dev
```

## Validar

```bash
npm run lint
npm run build
```
